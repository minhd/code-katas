# Roman Numerals

This program will output the Roman Numeral format for a given integer

## Specification

```
1 => I
2 => II
3 => III
4 => IV
5 => V
6 => VI
9 => IX
10 => X
11 => XI
15 => XV
40 => XL
41 => XLI
44 => XLIV
50 => L
100 => C
500 => D
900 => CM
1000 => M
```
Doesn't need to work above 5000

## Test
```
composer test
npm run dev
```

## Installation
```
composer install && npm install
```
